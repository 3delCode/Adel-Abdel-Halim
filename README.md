# البورتفوليو بتاعي — Adel Abdel Halim

أهلا — ده الموقع الشخصي بتاعي بيعرض شغلي، مشاريعي، وشهاداتي. اتعمل بــ HTML/CSS/JS وبسيط وسهل تعديله.

## إيه اللي في المشروع ده؟

- صفحة معرض شخصية فيها: الهيدر، قسم عني، المهارات، المشاريع، الشهادات، وصفحة تواصل.
- زر واتساب ثابت تحت على الشمال/اليمين (حسب طريقة العرض) عشان الناس تكلمك بسرعة.
- زر صغير للـ "العودة للأعلى" بيظهر لما تعمل سكرول وبيخليك ترجع فوق بسهولة.
- معمول بـ Bootstrap للأقسام وAOS للحركات الخفيفة.

## الملفات المهمة

- `index.html` — الصفحة الرئيسية.
- `style.css` — كل ستايلات الموقع.
- `main.js` — شوية جافاسكربت للتحكم في الثيم، السكربتات، وزر العودة للأعلى.
- `img/` — هنا الصور (البروفايل وصور المشاريع).

## إزاي أشغّله عندي بسرعة؟

لو عايز تجرب بسرعة ممكن تفتح `index.html` بالمتصفح مباشرة. لو عايز سيرفر محلي (أنسب لو هتعدل وتختبر روابط نسبية):

PowerShell (لو على ويندوز):

```powershell
# لو عندك Node.js وتحب تستخدم http-server
npm install -g http-server
http-server -c-1 . -p 8080
# افتح في المتصفح: http://localhost:8080
```

أو لو عندك Python:

```powershell
python -m http.server 8080
# افتح: http://localhost:8080
```

## عايز أغيّر رقم الواتساب؟

افتَح `index.html` و دور على السطر اللي فيه الرابط بتاع الواتساب. حاليا الرقم متظَبَّط كده:

```
https://wa.me/201155054433?text=...
```

لو عايز تستخدم الرقم اللي قلت عليه (01155054433) يبقى الشكل الدولي هو `201155054433` — يعني تحط الرقم الدولي من غير علامة `+`.

ولو عايز تغير الرسالة الجاهزة، عدّل النص بعد `?text=` (اتأكد تشفر الأحرف لو فيها مسافات/عربي).

## هرفع المشروع على GitHub — بسرعة

لو لسه ماعملتش ريبوزيتوري محلي، افتح PowerShell جوا فولدر المشروع واعمل:

```powershell
git init
git add .
git commit -m "Initial commit: portfolio"
```

بعد كده اخلق ريبوزيتوري جديد على GitHub (اسم زي `adel-portfolio`) وبعدين اربطه وادفع الكود:

```powershell
# استبدل <USERNAME> و <REPO> باسمك واسم الريبو
git remote add origin https://github.com/<USERNAME>/<REPO>.git
git branch -M main
git push -u origin main
```

لو عايز تنشره كـ GitHub Pages: روح لصفحة الريبو → Settings → Pages، اختار الفرع `main` والمجلد `/ (root)`، وبعدين Save. الموقع هيفضل متاح على:

```
https://<USERNAME>.github.io/<REPO>/
```
