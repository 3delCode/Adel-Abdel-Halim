# البورتفوليو بتاعي — Adel Abdel Halim

أهلا — ده الموقع الشخصي بتاعي بيعرض شغلي، مشاريعي، وشهاداتي. اتعمل بــ HTML/CSS/JS وبسيط وسهل تعديله.

## إيه اللي في المشروع ده؟

- صفحة معرض شخصية فيها: الهيدر، قسم عني، المهارات، المشاريع، الشهادات، وصفحة تواصل.
- زر واتساب ثابت تحت على الشمال/اليمين (حسب طريقة العرض) عشان الناس تكلمك بسرعة.
- زر صغير للـ "العودة للأعلى" بيظهر لما تعمل سكرول وبيخليك ترجع فوق بسهولة.
- معمول بـ Bootstrap للأقسام وAOS للحركات الخفيفة.
